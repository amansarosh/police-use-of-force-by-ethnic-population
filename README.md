# About and why

The data is a comprehensive overview of Washington State’s county-level demographics by race alongside police use-of-force incidents. Together, these datasets provide the necessary context to examine whether higher levels of police force are associated with more impoverished areas and whether racial composition within counties correlates with increased policing intensity. By combining demographic and policing data, we can identify potential patterns of systemic targeting or disproportionate enforcement across different communities. I think the data and project as a whole encompass everyone who lives within the state of Washington, as the data only addresses counties within Washington.
